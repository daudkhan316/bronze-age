# Bronze Age — start here

Browser RTS (Age of Empires II-inspired). TypeScript (strict) + Vite + HTML5
Canvas 2D + a small custom ECS, fixed-timestep deterministic sim. No game engine,
no backend. Built in **phases (0–6), one at a time** (stop + review after each).

**Read these first, every session — they are self-sufficient:**
[`docs/PLAN.md`](docs/PLAN.md) (plan + current status + architecture invariants +
how we work + the next phase's tasks) and [`docs/PROMPT.md`](docs/PROMPT.md) (the
original brief).

Commands: `npm run dev` (http://localhost:5173) · `npm run typecheck` · `npm run build`.
